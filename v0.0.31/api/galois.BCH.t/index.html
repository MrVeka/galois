    property galois.BCH.t : int

    The error-correcting capability of the code. The code can correct \(t\) bit errors in a codeword.

    Examples¶

    In [1]: bch = galois.BCH(15, 7); bch
Out[1]: <BCH Code: [15, 7, 5] over GF(2)>

In [2]: bch.t
Out[2]: 2
